# Tools

This directory contains personal notes on the tools I actively use during security work — offensive, defensive, and analytical.

## Why this directory exists

Tools are only as useful as your understanding of them. Running nmap or gobuster without knowing what the flags mean or why you are using them leads to missing obvious findings. This directory exists to build and maintain that understanding, not as a manual copy-paste but as a personal reference shaped by real use.

## What the need is

Every time I learn something new about a tool — a flag I misused, a use case I had not considered, a limitation I ran into — that knowledge disappears if not written down. These notes capture the practical layer: when to reach for a tool, what it actually does under the hood, and what to watch out for.

## What to expect

Each file focuses on a single tool or tool category. Entries cover:

- What the tool does and what problem it solves
- Common use cases and when to prefer it over alternatives
- Key flags and options I actually use, with context
- Gotchas, limitations, and things that tripped me up
- Example commands drawn from real engagements or labs

The focus is on the why and when — not just a list of commands. Tools covered include but are not limited to: nmap, gobuster, ffuf, Burp Suite, metasploit, tcpdump, linPEAS, and others as they come up.
