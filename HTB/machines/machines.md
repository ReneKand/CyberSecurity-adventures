# Machines

This directory contains write-ups for completed HTB machines — full compromises from initial enumeration through user and root flag capture.

## Why this directory exists

Machines are the core of HTB. Each one is a full system with a realistic attack surface. Documenting them creates a searchable reference for techniques I have used, patterns I have seen, and mistakes worth not repeating.

## What the need is

Without documentation, every machine becomes a one-time experience. Writing it down forces me to articulate what I actually did and why it worked. It also builds a record I can return to when a similar technique comes up on a different machine.

## What to expect

Each file covers a single machine and is named after it. The write-up includes:

- Initial enumeration output and interpretation
- Service-specific investigation and decisions made
- Exploitation approach and how it was reached
- Privilege escalation path with tool output where relevant
- Afterthoughts on what was learned and what could have been done differently

Credentials, flags, and usernames are redacted or generalized. The focus is on methodology, not copy-paste exploitation.
